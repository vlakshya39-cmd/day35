# Day 35 – Prompt Puzzle: Master AI Prompting Through Play

## 📌 Overview

Prompt Puzzle is an interactive HTML application designed to teach effective AI prompting through engaging gameplay. Players complete multiple prompt-engineering challenges by constructing, cleaning, and evaluating prompts while receiving instant feedback and a detailed performance report.

This project focuses on developing practical prompt engineering skills using a gamified learning experience.

---

## 🚀 Features

- 🧩 Interactive Prompt Puzzle gameplay
- 🏗️ Build the Best Prompt challenge
- 🧹 Clean the Prompt challenge
- ⭐ Choose the Best Prompt challenge
- 📊 Prompt Performance Report
- 🧬 Prompt DNA visualization
- 🎯 Accuracy and score tracking
- 💡 Hint system
- 🔄 Replay with randomized scenarios
- 📱 Responsive modern UI
- ⚡ Pure HTML, CSS and JavaScript (No frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---

## 🎮 Gameplay

The application contains multiple Software Development prompt-engineering scenarios where users must:

- Build optimized prompts
- Remove unnecessary prompt components
- Select the best prompt among multiple choices
- Improve prompt quality through practice
- Analyze performance using detailed feedback

---

## 📈 Learning Outcomes

Through this project, I learned:

- The importance of Role, Context, Task, and Output Format in prompts.
- How prompt quality directly affects AI-generated responses.
- How to distinguish weak, optimized, and over-engineered prompts.
- Best practices for prompt engineering in software development.
- Designing engaging educational web applications using vanilla JavaScript.

---


---

## ▶️ How to Run

1. Download the repository.
2. Open `prompt-puzzle.html` in any modern web browser.
3. Complete all prompt challenges.
4. Review your Prompt Performance Report.
5. Replay with different scenarios to improve your score.

---

## 📚 Key Takeaways

- Clear prompts outperform vague prompts.
- Overly complex prompts can reduce effectiveness.
- Structured prompting produces more reliable AI outputs.
- Gamification is an effective way to learn prompt engineering.
- Interactive web applications can be built efficiently using only HTML, CSS, and JavaScript.
